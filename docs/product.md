# product.md

## Product Overview

사용자와 자연스럽게 대화하며, 필요할 때는 도구를 호출해 실시간 정보나 작업까지 수행하는 AI 어시스턴트다. 단순 질의응답에 그치지 않고, 대화 맥락을 유지하면서 필요한 순간 외부 도구를 활용해 더 정확하고 실질적인 도움을 주는 것을 목표로 한다.

## Core Features

1. **멀티턴 대화**: 이전 대화 맥락을 유지하며 이어서 대화할 수 있다
2. **실시간 스트리밍 응답**: AI의 응답을 토큰 단위로 실시간으로 받아볼 수 있다
3. **도구 호출**: 필요한 순간 외부 도구를 호출해 그 결과를 반영한 응답을 받을 수 있다 (도구 목록은 추후 확정)
4. **사용자 인증**: 사용자별로 대화가 구분되고, 본인 대화에만 접근할 수 있다

## Product Terminology

- **어시스턴트 (Assistant)**: 이 프로젝트가 만드는 제품 자체를 가리키는 용어. AGENTS.md의 "AI 에이전트 서비스"는 이 제품이 속한 상위 카테고리를 뜻하고, 제품 자체를 지칭할 때는 "어시스턴트"로 통일한다.
- **대화 (Conversation)**: 사용자와 어시스턴트 사이의 하나의 연속된 멀티턴 교환 전체
- **메시지 (Message)**: 대화를 구성하는 개별 발화 단위 (사용자 메시지 또는 어시스턴트 메시지 하나)
- **도구 (Tool)**: 어시스턴트가 응답을 생성하기 위해 호출하는 외부 기능. ("함수 호출", "액션" 등 다른 표현 대신 "도구"로 통일)
- **세션 (Session)**: 인증된 사용자의 로그인 상태를 가리키는 용어로 한정한다 (대화 단위와 혼동하지 않도록)

## Roadmap / Next Steps

1. **단일 턴 대화 + 스트리밍 응답**: FastAPI 기본 골격을 세우고, async 처리와 SSE로 토큰 단위 응답 스트리밍부터 동작하게 한다
2. **멀티턴 대화**: DB에 대화/메시지를 저장하고, 이전 맥락을 포함해 다음 요청을 처리한다
3. **사용자 인증**: 사용자별로 대화를 구분하고, 본인 대화에만 접근하도록 한다
4. **도구 호출**: LLM이 도구 사용 여부를 판단하고, 실행 결과를 응답에 반영한다
